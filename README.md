# 🌱 Agroturismo Ecológico

Site desenvolvido para o **Concurso Agrinho 2026** — Subcategoria 3.

## 🎯 Objetivo do tema

Apresentar uma **pesquisa aprofundada** sobre **Turismo Rural, Agroturismo e Ecoturismo** como vetores de desenvolvimento sustentável no campo, conciliando:

- 🌿 **Sustentabilidade ambiental** (conservação e regeneração)
- 💰 **Sustentabilidade econômica** (diversificação de renda)
- 👥 **Sustentabilidade social** (combate ao êxodo rural)

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|------------|-----|
| **HTML5** | Estrutura semântica com tags `<header>`, `<main>`, `<section>`, `<nav>`, `<footer>` |
| **CSS3** | Variáveis CSS, Grid, Flexbox, `position: sticky`, Media Queries, animações `@keyframes` |
| **JavaScript (ES6+)** | IIFE, IntersectionObserver, `localStorage`, DOM API |
| **Google Fonts** | Inter (corpo) + Poppins (títulos) |

## 📁 Estrutura

```
buiu/

```

## ✨ Funcionalidades

- 🌗 **Modo escuro/claro/alto contraste** com persistência via `localStorage`
- 🔤 **Ajuste de fonte** (A- → A+ → A↺) com persistência
- 🌍 **Tradução PT/EN** automática
- 🧭 **Slider de 6 motores** com auto-advance, dots e pause no hover
- 📊 **Contadores animados** (42%, 150, 30) com easing
- 🎴 **Cards com tilt 3D** no hover do mouse
- 📜 **Scroll reveal** ao entrar no viewport
- ✉️ **Modal de contato** com validação de formulário
- ⌨️ **Atalhos de teclado** (Escape fecha modal, Tab navega)
- 📱 **Responsivo** (mobile, tablet, desktop)

## 🚀 Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Aguarde o loader (1.5s)
3. Use os botões da barra superior para mudar tema, fonte ou idioma
4. Role a página para ver as animações de scroll
5. Clique em "Receber por email" no hero para testar o modal

## 🧪 Compatibilidade

- ✅ Chrome / Edge / Firefox / Safari (versões 2023+)
- ✅ Mobile (iOS Safari, Android Chrome)
- ✅ Suporte a `prefers-reduced-motion` (acessibilidade)
- ✅ Suporte a `prefers-color-scheme` (modo escuro automático)
